# Movie Data Analytics Project

This project analyzes movie data using Python and Pandas to extract insights about ratings, box office performance, and genre trends.

## Features
- Data cleaning and preprocessing
- Genre-wise performance analysis
- Highest rated and highest earning movies
- Average box office comparison
- Visualizations using Matplotlib

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Identified top-performing genres
- Compared ratings and box office trends
- Visualized data using charts and scatter plots
